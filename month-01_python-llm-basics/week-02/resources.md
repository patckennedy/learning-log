## 💡 Helpful Tips
- Use `with open(filename, mode) as f:` to handle files safely (auto-closes file).  
- Remember:  
  - Lists = ordered, changeable  
  - Tuples = ordered, **immutable**  
  - Sets = unordered, unique items only  
  - Dicts = key–value pairs
